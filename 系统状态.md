```bash
cd backend
npm run seed
```

## ⚠️ 注意事项

1. **后端必须先启动**：前端依赖后端API
2. **数据库连接**：确保数据库服务器可访问
3. **端口占用**：
   - 前端：3000
   - 后端：3001
4. **API认证**：所有需要认证的API都需要JWT token
